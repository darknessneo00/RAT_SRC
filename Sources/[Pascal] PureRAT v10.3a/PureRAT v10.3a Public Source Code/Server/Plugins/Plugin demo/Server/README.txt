PureRAT plugins
===============

Little guide for PureRAT plugins server type.

How it works
============

Server plugins are separated in 2 parts: server part and client part.
Both parts need to be compiled by plugins builder and the client part needs to be
installed on remote machine. 

How to run client part
========================

First install server part in PureRAT server on local machine. Then load installed
plugin in Plugins bar of Client Manager.
