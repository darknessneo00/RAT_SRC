PureRAT plugins
===============

Delphi is the programming language use for these demos. It doesn't means that you can
not builds yours plugins in others programming languages. Plugins are native libraries 
and are just compiled with PureRAT plugins builder to work with program.

First of all compile these library file with a delphi compiler (Delphi 7 or higher),
then compile them with PureRAT plugins builder and... it's done!
